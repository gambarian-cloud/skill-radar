design note · multi-tenant

Tenant isolation is enforced at the Router layer. Each source connection carries a tenant_id claim; all downstream records are partitioned by it. The shared pipeline infrastructure never mixes tenant data — cluster computation, embeddings, and action dispatch are all scoped per tenant before touching storage.
