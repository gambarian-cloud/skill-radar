# Technique Catalog

Use 1 or 2 strong techniques per output. Do not stack many weak ones.

## Typography as the main move

Use when the content needs authority, clarity, or editorial feel.

- Pair a distinctive display face with a calm body face.
- Let one heading carry the visual identity.
- Use tighter width and stronger size contrast for more confidence.
- Create rhythm with spacing before adding decoration.

## Color as structure

Use when the interface feels flat or timid.

- Build a dominant neutral field with one sharp accent color.
- Use color to rank attention, not just decorate.
- Keep the palette small and repeat it consistently.
- Add one atmospheric background treatment instead of many small effects.

## Composition and depth

Use when the layout feels too safe or template-like.

- Introduce asymmetry where it improves flow.
- Use overlap, framing, or controlled negative space to add depth.
- Let one section break the grid on purpose.
- Use layering, shadow, texture, or border systems to create atmosphere.

## Motion and reveal

Use when the interface needs energy or polish.

- Prefer one orchestrated entrance sequence over scattered motion.
- Use motion to explain hierarchy and interaction, not to show off.
- Keep timing consistent and restrained.
- Respect reduced-motion preferences.

## Data and dashboards

Use when numbers need clarity and authority.

- Give one metric the strongest emphasis.
- Reduce chart clutter before adding visual flourish.
- Use annotation and grouping to tell the story of the numbers.
- Make labels, legends, and axes easier to parse than the default library output.

## Presentations and reports

Use when the output must be easy to scan from a distance.

- One main idea per slide or panel.
- Use fewer words and stronger structure.
- Let headings do more work than bullets.
- Favor one strong chart or visual over a busy collage.
