# Reference Library

Use these references as direction markers, not templates to copy.

## Product and interface references

- Stripe: strong hierarchy, elegant density, clear financial UI
- Linear: disciplined spacing, calm motion, sharp product focus
- Apple: restraint, typography, and premium clarity
- Airbnb: approachable clarity and warm visual systems

## Historical references

- Swiss design: structure, grid discipline, contrast, typographic clarity
- Bauhaus: bold reduction, shape discipline, function with form
- Editorial magazine layouts: hierarchy, pacing, image-text rhythm

## Direction choices

Choose one primary direction before polishing:

- Editorial
- Premium minimal
- Calm professional
- Bold product-marketing
- Playful modern
- Technical / developer-native
- Data-forward / dashboard-led

## Reference rule

Do not blend too many directions. Pick one primary lane, then one supporting influence.
